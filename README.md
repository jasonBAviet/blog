# Personal Blog

Blog ca nhan built with Next.js, TypeScript, Tailwind CSS, and Next Themes.

## Chay du an

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev`: chay moi truong development
- `npm run build`: build production
- `npm run start`: chay production build
- `npm run lint`: kiem tra lint

## Du lieu bai viet

- Noi dung bai viet lay tu thu muc `Source/`
- Ảnh bai viet duoc luu trong `public/source-images/`

## Ghi chu

- Trang web hien tai dung du lieu bai viet theo ngay va co ho tro font tieng Viet.
- Neu ban them bai moi, hay luu text theo dinh dang ngay va gan anh cung tien to ngay tuong ung.
